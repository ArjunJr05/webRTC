webRTC Project Documentation

Android WebRTC chat client with a Maven-based Spring WebSocket signaling backend. This site publishes the generated backend Java API documentation together with a simple project landing page.

Android App

Kotlin client using WebRTC DataChannel messaging with WebSocket signaling only for offer/answer/candidate exchange.

Backend

Spring Boot WebSocket signaling server that relays signaling JSON (join/offer/answer/candidate) between peers so they can establish a direct P2P WebRTC DataChannel connection.
